## わかったこと
- Procオブジェクトを作成
```
Proc.new
# procメソッド
proc { |a, b| a + b }

# ->構文
->(a, b) { a + b }
# lambdaメソッド
lambda { |a, b| a + b }

# ()を省略
-> a, b { a + b }
```
- バックトレースとは、プログラムが実行されてエラーが発生するまでのメソッド呼び出しの順番を示した情報のこと。上に行くほどエラーに近く、下に行くほどエラーからは遠くなる。
